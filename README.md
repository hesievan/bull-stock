## 指标体系 (v3.2: 6维度 16子指标)

### 估值维度 (20%)

| # | 指标 | 数据源 | 口径 |
|---|------|--------|------|
| 1 | PE历史分位 | tushare stock_daily.peTTM | 成分股口径，10年分位 |
| 2 | PB历史分位 | tushare stock_daily.pbMRQ | 成分股口径，10年分位 |
| 3 | 破净率(反向) | tushare stock_daily.pbMRQ | 全市场口径，反向计分 |
| 4 | ERP股权风险溢价 | index_daily_pe + bond_yield | 1/PE - 10年国债收益率 |

## 自动化

- `copaw cron` 每交易日16:30触发 `run_daily.py`
- 输出: JSON + 飞书通知(红区连续2天触发)
- 日报: MD + HTML(ECharts交互图) + PNG
