## 🟡 指标质量优化 — 传统牛市顶失真诊断 (全市场数据回测)

> 2026-06-01 回测发现: 全市场数据下，2015/2020/2021 传统牛市顶未触发红区(≥70)
> 2024 脉冲顶(73.5)准确触发 ✅ | 4个熊市底部全部命中绿区(≤35) ✅

### 失真根因拆解

**问题 1: 估值分位数被小盘股稀释 (最大拖后腿)**

全市场 5000+ 只股票中，小盘股 PE 普遍 100+，导致大票 PE=25 在全市场分位仅排 30-50 分位。

| 日期 | 状态 | PE分位(全市场) | 预期 | 差距 |
|------|------|--------------|------|------|
| 2015-06-12 | 牛市顶 | 99 ✅ | 99 | 0 |
| 2020-07-10 | 牛市启 | **48** | 80+ | **-32** |
| 2021-02-18 | 牛市顶 | **29** | 70+ | **-41** |
| 2024-10-08 | 脉冲顶 | 51 | 50 | 0 |

- [ ] **修复方案 A**: 改用市值加权 PE 中位数 (大票权重更高)
- [ ] **修复方案 B**: 仅计算沪深300+中证500成分股的 PE 分位 (排除小盘噪音)
- [ ] **修复方案 C**: PE 分位用 baostock 成分股计算，其余指标用全市场

**问题 2: `new_high_ratio` 永远为 0 (BUG)**

所有日期 `new_high_ratio=0`，技术维度少了一个强力信号。

| 日期 | 状态 | new_high_ratio | 预期 |
|------|------|---------------|------|
| 2015-06-12 | 牛市顶 | **0** | 80+ |
| 2020-07-10 | 牛市启 | **0** | 60+ |
| 2021-02-18 | 牛市顶 | **0** | 70+ |

- [ ] **根因**: `stock_daily` 只有 baostock 拉的成分股 K 线，创新高计算需要全市场当日最高价
- [ ] **修复方案 A**: 用全市场 daily 数据中的 high 字段计算创新高比例
- [ ] **修复方案 B**: 改为"距新高 5% 以内比例" (不严格要求创新高)

**问题 3: 2015 融资融券数据缺失**

2015 年 `margin_ratio=null`，资金维度少了 50% 权重。融资余额是 2015 杠杆牛的核心驱动。

| 日期 | 状态 | margin_ratio | northbound | 资金维度 |
|------|------|-------------|------------|---------|
| 2015-06-12 | 牛市顶 | **null** | 65 | 65 |
| 2020-07-10 | 牛市启 | 98 ✅ | 70 | 84 |
| 2021-02-18 | 牛市顶 | 89 ✅ | 75 | 82 |

- [ ] **修复方案**: 用 akshare `stock_margin_detail_szse` 补充 2010-2018 融资融券历史

**问题 4: 涨跌家数比反直觉 (A 股结构性牛市特征)**

2020-07-10 指数 3450 但涨跌家数比仅 26 — 多数股票下跌。

- [ ] 这是真实状况 (A 股牛市常是结构性的: 大票涨、小票跌)
- [ ] 但需在报告中注明: "涨跌家数比反映市场广度，结构性牛市中该指标偏低属正常"

## 🔧 故障排查

### 常见问题

| 症状 | 原因 | 解决 |
|------|------|------|
| `run_daily.py` 卡住不动 | baostock 登录超时 | `bs_logout()` 后重试，或重启进程 |
| tushare 报"频率超限" | 200次/分钟限制 | 等 1-2 分钟自动恢复；backfill 脚本已内置 sleep |
| 综合热度突然归零 | 某维度计算异常返回 None | 检查 `run_daily.log` 中的 ERROR |
| 飞书通知未收到 | Webhook URL 失效 | 检查 `json_writer.py` 中的 WEBHOOK_URL |
| stock_daily 股票数骤降 | tushare daily_basic 某日无数据 | 检查 `backfill_full_market.log` 中对应日期 |
| 前端不显示数据 | JSON 文件未生成 | 手动跑 `run_daily.py` 检查输出 |

### 数据修复

```bash
# 修复单日数据
python scripts/run_daily.py 2026-05-29

# 重新回填全市场 PE/PB (从指定日期)
python scripts/backfill_full_market_pe.py 2026-05-01

# 重新拉取 baostock 历史
python scripts/init_history.py 2015-01-01

# 重建 market_cap 表
python -c "
from src.data.fetcher import rebuild_market_cap
rebuild_market_cap()
"
```
