## ✅ 已完成 (2026-06-03)

### 全量历史回测 ✅
- 脚本: `scripts/backfill_history.py`
- 范围: 2015-01-05 ~ 2026-06-02, 2771 个交易日
- 耗时: ~12 小时 (1-2秒/天)
- 输出: `web/data/history_full.json` (501KB)
- 断点续传: 支持中断后从上次保存点继续

### 历史走势图 ✅
- 脚本: `scripts/gen_history_chart.py`
- 主图: 综合热度指数走势 + 红黄绿参考线 + 关键事件标注 + 极值标记
- 副图: 五维度走势对比 (可切换单维度/全部)
- 交互: 底部滑块缩放, 双图联动, 数据 Zoom
- 关键事件: 牛市顶(5178)、熔断底、贸易战底、新冠底、政策脉冲等
- 输出: `reports/history_chart.html` (1.1MB, 内联ECharts)

### AH溢价指数方案B ✅
- 替代已不可用的东方财富 push2his API
- 脚本: `scripts/ah_premium.py`
- 方案: akshare stock_hk_daily(H股) + baostock(A股) → 15只核心AH股溢价中位数
- 回写: SQLite `ah_premium` 表 (trade_date, premium, n_stocks)

### 资金维度异常修复 ✅
- `margin_ratio`: 从"rzmre绝对金额分位"改为"rzmre/amount比率分位"(504天窗口)
- `northbound`: 从"20日净买入天数占比"改为"净流入金额250日历史分位"
- 修复前: 资金维度 99.3 分(失真) → 修复后: 49.8 分(合理)

### 日报生成器 v2 ✅
- 脚本: `scripts/gen_report.py`
- 输出三份: MD + HTML(ECharts交互图) + PNG(精简信息图)
- 维度对比: 每个维度得分旁显示较昨日变化(↑/↓)
- 板块热度: TOP10 表格 + 龙头股
- 运行状态: 各 Step OK/FAILED/SKIPPED 状态

### 前端 SPA 重构 ✅
- `web/app.html`: 单页应用, 4 Tab (概览/拆解/板块/历史)
- ECharts 仪表盘 + 雷达图 + 历史走势 + 板块排名
- 数据加载全容错, 状态栏显示各 Step 状态

### 板块热度指数 ✅

## 🟢 日常维护

### 每日 (自动)
- `copaw cron` 每交易日 16:30 自动触发 `run_daily.py`
- 自动拉取: tushare 全市场K线/PE/PB/市值 → 融资融券/北向/国债
- 自动计算: 16子指标(加权) → 综合热度 → JSON 输出
- 自动通知: 红区(≥70) → 飞书推送（防抖: 连续2天）
