# A股牛市热度指数 — 量化策略

> 策略版本: v1.0 | 基于: v3.6热度指数 | 回测验证: 2015-2026

---

## 一、策略定位

**核心原则**: 仅提示离场/减仓，不发出进场或加仓信号。

**策略类型**: 被动择时 + 动态仓位管理

**适用场景**: 
- 沪深300/中证500等宽基指数基金
- 大盘蓝筹股组合
- 不适用于个股/小盘股/题材股

## 二、信号体系

### 2.1 热度信号

| 热度等级 | 分数 | 信号 | 行动 |
|---------|------|------|------|
| 🟢 绿色安全 | 0-40 | 安全区间 | 维持当前仓位 |
| 🟡 黄色警惕 | 40-55 | 关注信号 | 准备减仓 |
| 🟠 橙色关注 | 55-65 | 减仓信号 | 分批减仓 |
| 🔴 红色预警 | 65+ | 清仓信号 | 大幅减仓/清仓 |

### 2.2 维度信号

| 维度 | 阈值 | 含义 | 行动建议 |
|------|------|------|---------|
| 估值 | ≥80 | 极度高估 | 优先减仓 |
| 资金 | ≥80 | 杠杆过高 | 关注风险 |
| 情绪 | ≥80 | 情绪过热 | 准备离场 |
| 技术 | ≥80 | 严重超买 | 短期风险 |

### 2.3 组合信号

**最强减仓信号** (满足任意2项):
1. 综合热度 ≥65 (红色)
2. 估值维度 ≥80 + 情绪维度 ≥80
3. 连续3天橙色以上

**清仓信号** (满足任意1项):
1. 综合热度 ≥70 且 估值维度 ≥90
2. 综合热度连续5天 ≥65

## 三、仓位管理

### 3.1 基准仓位

| 市场状态 | 基准仓位 | 说明 |
|---------|---------|------|
| 低估值 (PE分位<30%) | 90-100% | 长期配置 |
| 正常估值 (30-60%) | 70-80% | 标准配置 |
| 高估值 (60-80%) | 50-60% | 减仓配置 |
| 极端高估 (>80%) | 20-30% | 防御配置 |

### 3.2 热度调整

根据综合热度动态调整仓位:

```
目标仓位 = 基准仓位 × 热度调整系数

热度调整系数:
- 热度 < 30:  1.0 (满仓)
- 热度 30-50: 0.9
- 热度 50-60: 0.8
- 热度 60-70: 0.6
- 热度 > 70:  0.3
```

### 3.3 实际仓位计算

```python
def calculate_target_position(heat_score, pe_percentile):
    """
    计算目标仓位
    
    Args:
        heat_score: 综合热度 (0-100)
        pe_percentile: PE历史分位 (0-100)
    
    Returns:
        target_position: 目标仓位 (0-100%)
    """
    # 基准仓位 (基于估值)
    if pe_percentile < 30:
        base = 95
    elif pe_percentile < 60:
        base = 75
    elif pe_percentile < 80:
        base = 55
    else:
        base = 25
    
    # 热度调整
    if heat_score < 30:
        factor = 1.0
    elif heat_score < 50:
        factor = 0.9
    elif heat_score < 60:
        factor = 0.8
    elif heat_score < 70:
        factor = 0.6
    else:
        factor = 0.3
    
    target = base * factor
    return max(20, min(100, target))  # 最低20%仓位
```

## 四、交易规则

### 4.1 减仓规则

**触发条件**: 综合热度 ≥55 (橙区) 或 估值维度 ≥80

**减仓节奏**:
- 第1天进入橙区: 减仓10%
- 连续2天橙区: 再减仓10%
- 进入红区: 一次性减仓至目标仓位

**示例**:
```
初始仓位: 80%
第1天: 热度58 → 减仓10% → 仓位70%
第2天: 热度62 → 再减10% → 仓位60%
第3天: 热度68 → 进入红区 → 减至目标40%
```

### 4.2 加仓规则

**触发条件**: 综合热度 ≤35 (绿区) 且 估值分位 ≤40

**加仓节奏**:
- 每下跌5分热度: 加仓5%
- 单次最多加仓10%
- 不追涨: 热度>50时禁止加仓

**示例**:
```
当前仓位: 60%, 热度40
热度跌至35: 加仓5% → 仓位65%
热度跌至30: 加仓5% → 仓位70%
热度回升至45: 停止加仓
```

### 4.3 止损规则

**硬止损**: 仓位亏损超过15%时强制减仓至50%

**热度止损**: 
- 热度从<40快速升至>65 (3天内): 减仓50%
- 连续5天红色: 清仓至20%

## 六、风险提示

### 6.1 策略局限

| 局限 | 说明 | 应对 |
|------|------|------|
| 滞后性 | 热度指标有2-3天滞后 | 不追涨杀跌 |
| 假信号 | 单日波动可能触发错误信号 | 防抖机制(连续2天确认) |
| 极端行情 | 政策驱动型行情可能失效 | 结合基本面判断 |
| 个股不适用 | 仅适用于宽基指数 | 不用于个股交易 |

### 6.2 使用建议

1. **长期持有为主**: 策略目的是优化买入卖出时点，不是频繁交易
2. **分批操作**: 减仓/加仓分2-3次完成，避免一次性操作
3. **结合基本面**: 热度指标仅作为参考，需结合宏观经济判断
4. **保持纪律**: 严格执行信号，不因情绪改变策略
