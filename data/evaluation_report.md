## 二、逐指标评估

### 估值维度 (6个指标)

#### 1. PE 历史分位 ✅ 可靠

| 属性 | 详情 |
|------|------|
| **数据源** | baostock `query_history_k_data_plus` → peTTM |
| **获取方式** | 成分股个股 PE 中位数，10年历史分位 |
| **真实性** | ✅ **真实数据**。baostock 提供的 peTTM 是基于最近4个季度净利润计算的真实滚动市盈率 |
| **覆盖度** | 2015~2026 完整，PE 有效率 92% |
| **回测表现** | 2015牛市顶 98分 ✅，2018熊底 1分 ✅，2021牛市顶 77分 ✅，2024脉冲顶 32分 ⚠️ |
| **问题** | 仅基于~260只成分股（沪深300+上证50+中证500），非全市场。对中小盘覆盖不足 |

**回测验证**：PE 分位在 2015/2021 两个牛市顶部均超过 77 分，熊市底部均低于 5 分。**区分度优秀**。但 2024 脉冲反弹仅 32 分，说明快速脉冲行情中 PE 来不及反应。

#### 2. PB 历史分位 ✅ 可靠

| 属性 | 详情 |
|------|------|
| **数据源** | baostock → pbMRQ |
| **真实性** | ✅ **真实数据**。基于最近季度净资产计算的滚动市净率 |
| **回测表现** | 2015顶 95分 ✅，2018底 0分 ✅，2021顶 86分 ✅ |
| **问题** | 同 PE，仅成分股样本 |

#### 3. 股债性价比 ERP ⚠️ 有条件可靠

| 属性 | 详情 |
|------|------|
| **数据源** | baostock PE中位数 + tushare 国债收益率 (yc_cb) |
| **真实性** | ✅ 真实数据。ERP = 1/PE中位数 - 10Y国债收益率 |
| **覆盖缺陷** | ⚠️ **bond_yield 仅从 2018-12-30 开始**。2015~2018 年默认使用 2.5% 作为国债收益率替代值 |
| **回测表现** | 2015顶 99分 ✅，2018底 2分 ✅（但用了默认2.5%，结果可信度降低） |
| **风险** | 2015年真实10Y国债约3.5%，默认2.5%导致ERP被高估，分位偏高 |

#### 4. 破净率 ✅ 可靠但需注意解读

| 属性 | 详情 |
|------|------|
| **数据源** | baostock pbMRQ < 1 的股票占比 |
| **真实性** | ✅ **真实数据** |
| **回测表现** | 2015顶 13分（低破净=贵）✅，2018底 100分（高破净=便宜）✅ |
| **注意** | 该指标在代码中是**反向**的：分数高 = 破净多 = 市场便宜。需在报告中明确说明 |

#### 5. 巴菲特指标 ✅ 可靠

| 属性 | 详情 |
|------|------|
| **数据源** | akshare M2月度 + tushare daily_basic 全市场总市值 |
| **真实性** | ✅ **真实数据**。M2来自央行公布数据，总市值来自tushare聚合 |
| **覆盖** | M2 2008~2026 完整，总市值 2015~2026 完整 |
| **回测表现** | 2015顶 100分 ✅，2018底 1分 ✅，2021顶 97分 ✅ |
| **单位修正** | 已修正：total_mv(万元) ÷ 10000 → 亿元，再与 M2(亿元) 做比 |
| **可靠性** | ⭐ **估值维度中最可靠的指标** — 数据源权威，覆盖完整 |

#### 6. 沪深300股债比 ⚠️ 有条件可靠

| 属性 | 详情 |
|------|------|
| **数据源** | tushare index_pe_history (HS300 pe_ttm) + bond_yield |
| **真实性** | ✅ 真实数据。E/P = 1/PE_ttm |
| **覆盖缺陷** | ⚠️ bond_yield 仅从 2018-12 开始，2019前无数据 |
| **回测表现** | 2020顶 99分 ✅，2021顶 100分 ✅，2022底 0分 ✅ |
| **问题** | 2015/2018 测试中为 N/A |

### 资金维度 (2个指标)

#### 7. 融资买入占比 ⚠️ 有条件可靠

| 属性 | 详情 |
|------|------|
| **数据源** | tushare `margin` (rzye/rzmre) |
| **真实性** | ✅ **真实数据**。来自交易所官方融资融券汇总 |
| **覆盖缺陷** | ⚠️ **仅从 2019-10-21 开始**。2019年前 N/A |
| **计算方法** | rzmre(当日融资买入额) / 全市场成交额 |
| **回测表现** | 2020顶 98分 ✅，2024脉冲 100分 ✅，2022底 24分 ✅ |
| **问题** | 2015牛市无数据，该轮牛市最重要的特征（杠杆资金）无法捕捉 |

#### 8. 北向资金方向 ⚠️ 精度不足

| 属性 | 详情 |
|------|------|
| **数据源** | tushare `moneyflow_hsgt` (hgt/sgt) |
| **真实性** | ✅ **真实数据**。来自交易所官方北向资金流水 |
| **覆盖** | 2015~2026 完整（2,682行） |
| **计算方法** | ⚠️ **仅用近20日净买入天数占比**（二值化：正=1，负=0） |
| **问题** | 将资金流入简化为"天数占比"，丢失了**金额量级**信息。连续20天小额净买入 = 100分，单日巨额净买入也被同等对待 |
| **回测** | 2024脉冲 100分 ✅，但2015顶仅 65分 ⚠️ |

### 技术维度 (4个指标)

#### 14. 站上年线比例 ✅ 可靠

| 属性 | 详情 |
|------|------|
| **数据源** | baostock close vs 250日均值 |
| **真实性** | ✅ **真实数据** |
| **回测** | 2015顶 89分 ✅，2018底 12分 ✅，区分度好 |

#### 15. 创新高占比 ❌ 失效

| 属性 | 详情 |
|------|------|
| **数据源** | baostock close vs 250日最高 |
| **问题** | ⚠️ **所有测试日期均为 0 分** |
| **原因** | 成分股（大盘蓝筹）在近10年中很少创250日新高。数据库仅~260只股票，且大盘股创新高频率低 |
| **结论** | **该指标在当前数据条件下完全失效，建议移除或替换** |

#### 16. 均线偏离度 ✅ 可靠

| 属性 | 详情 |
|------|------|
| **数据源** | 上证综指 close / MA250 - 1 |
| **真实性** | ✅ **真实数据** |
| **回测** | 2015顶 N/A（MA250数据不足），2020顶 100分 ✅，2024脉冲 100分 ✅ |
| **问题** | 2015年因 lookback 窗口限制返回 N/A |

#### 17. 量价背离 ❌ 精度严重不足

| 属性 | 详情 |
|------|------|
| **数据源** | 上证综指 价格趋势 vs 量比 |
| **问题** | ⚠️ **仅输出 35/50/65 三个固定值**，所有测试日期均为 50 或 65 |
| **原因** | 二值化判断（price_up × vol_shrink），非连续评分 |
| **结论** | **该指标几乎无区分度，建议重写为连续评分** |

## 六、优化建议（按优先级）

### P0 — 必须修复

1. **补充 2010-2018 国债收益率历史数据**（akshare 有更长历史）
2. **补充 2015-2019 融资融券历史数据**（akshare `stock_margin_detail_szse/sse`）
3. **移除或替换"创新高占比"** — 改为"距新高5%以内比例"或直接删除
4. **重写"量价背离"** — 从二值化改为连续评分（如10日量价相关系数的分位）

### P1 — 建议修复

5. **北向资金改为金额分位** — 用净流入金额的10年分位替代天数占比
6. **行业分化度改为连续分位** — 用10年std分位替代4档静态阈值
7. **扩大成分股样本** — 将 stock_daily 扩展到全市场（用tushare daily_basic的PE/PB）
8. **考虑降低红区阈值至 65** — 当前70导致2015/2021牛市顶部漏报

### P2 — 优化

9. **新增投资者数据自动更新** — 接入中国结算API或定期爬取
10. **AH溢价数据补充** — 等 Clash TUN 问题解决后用 akshare 补全
11. **回测自动化** — 将 backtest.py 纳入 CI，每次代码变更自动验证

## 七、结论

### 做得好的
- ✅ **所有指标均使用真实数据**，无估算或替代值
