## 二、调整方案

### 方案概览

**从 v3.0 (16子指标) → v3.1 (16子指标)**

| 操作 | 指标 | 理由 |
|------|------|------|
| 🔄 替换 | 巴菲特指标 → ERP | ERP是更好的估值指标 |
| 🔄 替换 | 换手率(绝对) → 自由流通换手率 | 避免IPO扩容失真 |
| 🔄 替换 | 北向资金(2项) → 北向资金(1项) | 文档评级低相关 |
| 🔄 替换 | MA250站上比 → MA排列比 | 更精细的趋势判断 |
| 🆕 新增 | M1-M2剪刀差 | 宏观流动性领先指标 |
| 🆕 新增 | 新发基金规模 | 散户增量资金滞后指标 |
| ❌ 删除 | 巴菲特指标 | 被ERP替代 |
| ❌ 删除 | 北向累计流入分位 | 与北向方向高度相关，冗余 |

### 2.1 维度权重调整

| 维度 | v3.0 | v3.1 | 理由 |
|------|------|------|------|
| 估值 | 25% | 20% | 新增宏观维度后，估值权重适度降低 |
| 宏观 | 0% | **15%** | 文档强调流动性是牛市发电机 |
| 资金 | 25% | 20% | 北向降权后，资金维度适度降低 |
| 情绪 | 20% | 20% | 保持不变 |
| 技术 | 10% | 10% | 保持不变 |
| 结构 | 20% | **15%** | 文档不重视结构，适度降低 |

**总权重**: 20% + 15% + 20% + 20% + 10% + 15% = **100%**

### 2.2 新增指标详情

#### 2.2.1 M1-M2增速剪刀差（宏观维度）

| 项目 | 内容 |
|------|------|
| **公式** | `剪刀差 = M1同比增速 - M2同比增速` |
| **数据来源** | tushare `macro_china_money_supply` (m1_yoy, m2_yoy) |
| **计算逻辑** | 月频数据，插值到日频。剪刀差由负转正=牛市启动信号，正→负=牛市见顶信号 |
| **时间范围** | 近10年 |
| **代表意义** | M1反映企业活期资金（随时准备投资），M2反映广义货币。剪刀差扩大=资金活期化=牛市最强风向标 |
| **数据量** | 月频220行，插值到日频 |
| **获取难度** | 低 — tushare已覆盖 |

#### 2.2.2 M2同比（宏观维度）

| 项目 | 内容 |
|------|------|
| **公式** | `M2同比增速` |
| **数据来源** | tushare `macro_china_money_supply` (m2_yoy) |
| **计算逻辑** | 月频数据，插值到日频。M2同比持续回升=流动性宽松=牛市条件 |
| **时间范围** | 近10年 |
| **代表意义** | 流动性总量指标，M2同比回升通常领先股市3-6个月 |
| **数据量** | 月频220行 |
| **获取难度** | 低 — tushare已覆盖 |

#### 2.2.3 股权风险溢价ERP（估值维度）

| 项目 | 内容 |
|------|------|
| **公式** | `ERP = 1/PE - 10年国债收益率` |
| **数据来源** | 已有数据可算：index_daily_pe.pe_med + bond_yield.yield_rate |
| **计算逻辑** | ERP触及历史+2σ=大底信号，跌破-1σ=牛市狂热 |
| **时间范围** | 近10年 |
| **代表意义** | 衡量股票资产相对于无风险债券的性价比，A股见底/见顶的核心指标 |
| **数据量** | 2750行(index_daily_pe) |
| **获取难度** | 低 — 用现有数据即可计算 |

#### 2.2.4 新发偏股型基金规模（情绪维度）

| 项目 | 内容 |
|------|------|
| **公式** | `新发基金规模(亿元/月)` |
| **数据来源** | 需要外部数据源（Wind/Choice），或tushare基金接口 |
| **计算逻辑** | 月频数据。新发基金规模天量=基民疯狂入场=指数终极大顶（滞后指标） |
| **时间范围** | 近10年 |
| **代表意义** | 散户增量资金的代理变量，见顶信号 |
| **数据量** | 月频120行 |
| **获取难度** | 中 — 需要基金发行数据 |

### 2.3 修改指标详情

#### 2.3.1 修正换手率口径

| 项目 | v3.0 | v3.1 |
|------|------|------|
| 公式 | `成交额 / 全市场流通市值` | `成交额 / 自由流通市值` |
| 数据源 | stock_daily.circ_mv | stock_daily.circ_mv (需确认口径) |
| 问题 | circ_mv是总流通市值，不是自由流通市值 | 文档强调用自由流通市值 |

#### 2.3.2 修正北向资金权重

| 项目 | v3.0 | v3.1 |
|------|------|------|
| 指标数 | 2项(方向+累计流入) | 1项(累计流入分位) |
| 权重 | 25%资金维度中占2/3 | 20%资金维度中占1/3 |
| 理由 | 两项高度相关(r>0.9)，冗余 | 保留累计流入分位(更稳定) |

#### 2.3.3 替换技术维度指标

| 项目 | v3.0 | v3.1 |
|------|------|------|
| 指标 | MA250站上比 | MA排列比(MA20>MA60>MA120) |
| 理由 | MA250过于滞后 | MA排列更灵敏，牛市中后期也能捕捉 |
