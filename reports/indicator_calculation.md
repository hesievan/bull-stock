# A股牛市热度指数 — 指标计算说明

> 版本: v3.3 | 6维度 16子指标

---

## 一、估值维度 (权重 25%)

### 1. PE历史分位

| 项目 | 内容 |
|------|------|
| **计算公式** | `PE分位 = (历史PE中位数 < 当前PE中位数) / 历史样本数 × 100` |
| **数据来源** | tushare `stock_daily.peTTM` (全市场) + `index_daily_pe` (预计算表) |
| **计算逻辑** | 1) 取当日沪深300+中证500成分股的PE值<br>2) 过滤PE∈(0,500]的异常值<br>3) 计算成分股PE中位数<br>4) 与历史2753天的PE中位数比较分位 |
| **代表意义** | PE分位高=估值贵=牛市特征 |
| **当前值** | PE中位数=25.78, 分位=58.4% |

### 2. PB历史分位

| 项目 | 内容 |
|------|------|
| **计算公式** | `PB分位 = (历史PB中位数 < 当前PB中位数) / 历史样本数 × 100` |
| **数据来源** | tushare `stock_daily.pbMRQ` + `index_daily_pe` (预计算表) |
| **计算逻辑** | 同PE，成分股口径PB中位数的历史分位 |
| **代表意义** | PB分位高=估值贵=牛市特征 |
| **当前值** | PB中位数=2.20, 分位=61.1% |

### 3. 破净率 (反向)

| 项目 | 内容 |
|------|------|
| **计算公式** | `破净率 = PB<1的股票数 / 有PB数据的总股票数` |
| **数据来源** | tushare `stock_daily.pbMRQ` + `daily_below_net` (预计算表) |
| **计算逻辑** | 1) 全市场统计PB<1的股票占比<br>2) 与历史2772天的破净率比较分位<br>3) **反向计分**: 破净率高=便宜=低分 |
| **代表意义** | 破净率高=市场便宜=低分；破净率低=市场贵=高分 |
| **当前值** | 破净率=7.77%, 分位=36.2%(反向) |

### 4. ERP股权风险溢价

| 项目 | 内容 |
|------|------|
| **计算公式** | `ERP = 1/PE - 10年国债收益率` |
| **数据来源** | `index_daily_pe.pe_med` (PE中位数) + `bond_yield.yield_rate` (10年国债) |
| **计算逻辑** | 1) 股票收益率 = 1/PE × 100%<br>2) ERP = 股票收益率 - 国债收益率<br>3) 与历史1797天的ERP比较分位<br>4) **反向计分**: ERP高=股票便宜=低分 |
| **代表意义** | ERP高=股票性价比高=低分；ERP低=股票贵=高分 |
| **当前值** | ERP=2.19% (股票收益率3.93% - 国债1.74%) |

## 二、宏观维度 (权重 15%)

### 5. M1-M2增速剪刀差

| 项目 | 内容 |
|------|------|
| **计算公式** | `剪刀差 = M1同比增速 - M2同比增速` |
| **数据来源** | akshare `macro_china_money_supply` → `daily_macro` (日频) |
| **计算逻辑** | 1) 从月频M1/M2数据插值到日频<br>2) 计算剪刀差 = M1同比 - M2同比<br>3) 与历史2751天的剪刀差比较分位 |
| **代表意义** | 剪刀差正且扩大=资金活期化=牛市信号 |
| **当前值** | 剪刀差=-3.6 (M1同比5.0% - M2同比8.6%) |

### 6. M2同比增速

| 项目 | 内容 |
|------|------|
| **计算公式** | `M2同比增速` |
| **数据来源** | akshare `macro_china_money_supply` → `daily_macro` (日频) |
| **计算逻辑** | 1) 从月频M2数据插值到日频<br>2) 与历史2751天的M2同比比较分位 |
| **代表意义** | M2同比回升=流动性宽松=牛市条件 |
| **当前值** | M2同比=8.6% |

## 三、资金维度 (权重 15%)

### 7. 北向资金20日累计流入分位

| 项目 | 内容 |
|------|------|
| **计算公式** | `分位 = (历史250日窗口中20日累计 < 当前20日累计) / 窗口数 × 100` |
| **数据来源** | tushare `northbound_history.north_net` |
| **计算逻辑** | 1) 取最近20日累计净流入<br>2) 计算历史250日各窗口的20日累计流入<br>3) 当前值在历史中的分位 |
| **代表意义** | 分位高=外资持续流入=看好A股=高分 |
| **当前值** | 近20日累计=7819万, 分位=99.6% (历史最高) |

### 8. 融资余额占流通市值比

| 项目 | 内容 |
|------|------|
| **计算公式** | `比值 = 融资余额rzye / 全市场流通市值total_circ_mv` |
| **数据来源** | tushare `margin_history` + `daily_circ_mv` (预计算表) |
| **计算逻辑** | 1) 专注于融资余额(融券余额相对极小)<br>2) 3年历史窗口(750日)<br>3) rzye单位元, circ_mv单位万元, 需转换<br>4) 阈值: <2.5%温和, >4.0%过热 |
| **代表意义** | 比值高=杠杆高=牛市特征 |
| **当前值** | 2.80%, 分位=99.9% (历史最高) |

## 四、情绪维度 (权重 20%)

### 9. 换手率

| 项目 | 内容 |
|------|------|
| **计算公式** | `换手率 = 全市场成交额(千元) / 全市场流通市值(万元) × 10` |
| **数据来源** | tushare `stock_daily.amount` + `stock_daily.circ_mv` + `daily_turnover` (预计算表) |
| **计算逻辑** | 1) 注意: amount单位千元, circ_mv单位万元<br>2) 仅用tushare amount日期(>4000行)<br>3) 与历史1374天比较分位 |
| **代表意义** | 换手率高=市场活跃=情绪高涨 |
| **当前值** | 换手率=3.95%, 分位=99.8% |

### 10. 涨跌家数比

| 项目 | 内容 |
|------|------|
| **计算公式** | `比值 = 上涨家数 / 下跌家数` |
| **数据来源** | tushare `stock_daily.pct_change` + `daily_updown` (预计算表) |
| **计算逻辑** | 1) 统计当日上涨/下跌股票数量<br>2) 计算比值<br>3) 与历史2772天比较分位 |
| **代表意义** | 比值高=普涨=牛市特征 |
| **当前值** | 比值=0.37, 分位=19.6% |

### 11. 涨停占比

| 项目 | 内容 |
|------|------|
| **计算公式** | `占比 = 涨停股票数(pct_change≥9.9%) / 有数据总股票数` |
| **数据来源** | tushare `stock_daily.pct_change` + `daily_limit` (预计算表) |
| **计算逻辑** | 1) 统计涨停股票占比<br>2) 与历史2772天比较分位 |
| **代表意义** | 占比高=极端乐观=情绪过热 |
| **当前值** | 103/5192=1.98%, 分位=79.2% |

### 12. 涨跌停比

| 项目 | 内容 |
|------|------|
| **计算公式** | `比值 = 涨停数 / max(跌停数, 1)` (封顶10) |
| **数据来源** | tushare `stock_daily.pct_change` + `daily_limit` (预计算表) |
| **计算逻辑** | 1) 涨停数/跌停数<br>2) 与历史2772天比较分位 |
| **代表意义** | 比值高=涨停远多于跌停=情绪极度乐观 |
| **当前值** | 103/13=7.92, 分位=93.2% |

## 五、技术维度 (权重 10%)

### 13. MA排列比

| 项目 | 内容 |
|------|------|
| **计算公式** | `比例 = MA20 > MA60 > MA120 的股票占比` |
| **数据来源** | tushare `stock_daily.close` + `daily_ma_alignment` (预计算表) |
| **计算逻辑** | 1) 计算每只股票的MA20/MA60/MA120<br>2) 统计三均线多头排列的股票占比<br>3) 与历史2713天比较分位 |
| **代表意义** | 比例高=多数股票趋势向上=牛市特征 |
| **当前值** | 38.84%, 分位=43.6% |

### 14. 均线偏离度

| 项目 | 内容 |
|------|------|
| **计算公式** | `偏离度 = (上证综指收盘价 / MA250 - 1) × 100` |
| **数据来源** | tushare `index_daily.sh000001` |
| **计算逻辑** | 1) 计算上证综指偏离250日均线的百分比<br>2) 与历史2771天比较分位 |
| **代表意义** | 偏离大=价格远高于均线=泡沫风险 |
| **当前值** | 偏离度=3.84%, 分位=67.8% |

## 六、结构维度 (权重 15%)

### 15. 行业分化度 (连续版, 反向)

| 项目 | 内容 |
|------|------|
| **计算公式** | `分化度 = 各行业平均涨跌幅的标准差(std)` |
| **数据来源** | tushare `stock_daily.pct_change` + `stock_industry.industry` + `daily_sector_div` (预计算表) |
| **计算逻辑** | 1) 按84个行业分组计算平均涨跌幅<br>2) 取标准差<br>3) 与历史2772天的月度std比较分位<br>4) **反向计分**: std低=普涨=高分 |
| **代表意义** | 分化度低=普涨=全面牛市=高分 |
| **当前值** | std=1.24, 分位=42.8%(反向) |

### 16. AH溢价指数

| 项目 | 内容 |
|------|------|
| **计算公式** | 按5个区间赋分(月频数据) |
| **数据来源** | 恒生沪深港通AH股溢价指数历史数据(CSV) → `ah_premium_monthly` |
| **计算逻辑** | 1) 日频数据转月频(取每月最后一个交易日)<br>2) 按5个区间赋分:<br>   - <98: 10分(极致低估)<br>   - 98-115: 30分(偏低)<br>   - 115-135: 50分(中性)<br>   - 135-145: 70分(偏高)<br>   - >145: 90分(极致泡沫) |
| **代表意义** | AH溢价高=A股相对H股贵=牛市特征 |
| **当前值** | AH=119.45, 分位=50分(中性区间) |

## 七、综合得分计算

### 7.1 维度内合成

每个维度内的子指标**等权合成**。若某子指标异常(>3σ)或为0/None则舍弃，其余重新等权归一。

### 7.2 维度间加权合成

```
综合得分 = 估值×25% + 宏观×15% + 资金×15% + 情绪×20% + 技术×10% + 结构×15%
```

### 7.3 热度区间

| 颜色 | 分数 | 含义 |
|------|------|------|
| 🟢 绿色安全 | 0–40 | 估值合理/偏低，情绪冷淡 |
| 🟡 黄色警惕 | 40–65 | 部分指标偏高，需关注 |
| 🔴 红色预警 | 65–100 | 多项指标历史高位，考虑减仓 |

### 7.4 防抖机制

- 红区连续2天才发"进入红区"飞书通知
- 恢复1天后发"脱离红区"通知
