# A股牛市热度指数 — 指标详细说明

> 更新时间: 2026-06-04 | 指标版本: v3.0 (5维度 16子指标, 加权合成)
> 数据覆盖: 2015-01-05 ~ 2026-06-03 (2772个交易日)
> 数据源: tushare(全市场K线/PE/PB/市值/融资融券/北向) + akshare(M2/AH溢价)

---

## 一、指标体系总览

| 维度 | 权重 | 子指标数 | 子指标 |
|------|------|---------|--------|
| 估值 | 25% | 4 | PE分位, PB分位, 破净率, 巴菲特指标 |
| 资金 | 25% | 3 | 两融余额比, 北向资金方向, 北向累计流入分位 |
| 情绪 | 20% | 4 | 换手率, 涨跌家数比, 涨停占比, 涨跌停比 |
| 技术 | 10% | 2 | 站上年线比例, 均线偏离度 |
| 结构 | 20% | 2 | 行业分化度(连续版), AH溢价指数 |

**综合得分 = 估值×25% + 资金×25% + 情绪×20% + 技术×10% + 结构×20%**

## 二、各指标详细说明

### 估值维度 (权重25%)

#### 1. PE历史分位

| 项目 | 内容 |
|------|------|
| **名称** | PE历史分位 (PE_percentile) |
| **计算公式** | `PE分位 = (历史PE中位数 < 当前PE中位数) / 历史样本数 × 100` |
| **时间范围** | 近10年 (2015-01-30 ~ 2026-06-03) |
| **数据来源** | tushare stock_daily.peTTM (全市场) + index_daily_pe (预计算表) |
| **计算逻辑** | 沪深300+中证500成分股口径，用历史成分股截面(月末)计算PE中位数，查预计算表index_daily_pe |
| **代表意义** | 衡量当前PE在历史上的相对位置。高分=估值偏贵=牛市顶峰特征 |
| **数据量** | 预计算表: 2753天; 覆盖719只成分股/天 |
| **当前值** | PE中位数=25.78, 分位=58.4% |
| **分层含义** | 0-20%: 极度低估(熊底) \| 20-40%: 偏低 \| 40-60%: 中性 \| 60-80%: 偏高 \| 80-100%: 极度高估(牛市顶) |

#### 2. PB历史分位

| 项目 | 内容 |
|------|------|
| **名称** | PB历史分位 (PB_percentile) |
| **计算公式** | `PB分位 = (历史PB中位数 < 当前PB中位数) / 历史样本数 × 100` |
| **时间范围** | 近10年 |
| **数据来源** | tushare stock_daily.pbMRQ (全市场) + index_daily_pe (预计算表) |
| **计算逻辑** | 同PE，成分股口径PB中位数的历史分位 |
| **代表意义** | 衡量当前PB在历史上的相对位置。与PE互补，PB更稳定 |
| **数据量** | 预计算表: 2753天; 覆盖733只成分股/天 |
| **当前值** | PB中位数=2.20, 分位=61.1% |
| **分层含义** | 0-20%: 极度低估 \| 20-40%: 偏低 \| 40-60%: 中性 \| 60-80%: 偏高 \| 80-100%: 极度高估 |

#### 3. 破净率 (反向)

| 项目 | 内容 |
|------|------|
| **名称** | 破净率 (below_net_rate) |
| **计算公式** | `破净率 = PB<1的股票数 / 有PB数据的总股票数` |
| **时间范围** | 近10年 (2015-01-05 ~ 2026-06-03) |
| **数据来源** | tushare stock_daily.pbMRQ (全市场) + daily_below_net (预计算表) |
| **计算逻辑** | 全市场口径，统计PB<1的股票占比。反向计分：破净率越高=市场越便宜=得分越低 |
| **代表意义** | 破净率高=市场便宜=低分；破净率低=市场贵=高分 |
| **数据量** | 预计算表: 2772天; 最新: 7.77% (402/5173只) |
| **当前值** | 破净率=7.77%, 分位=36.2%(反向) |
| **分层含义** | 0-20%: 破净率高(便宜, 低分) \| 40-60%: 中性 \| 80-100%: 破净率低(贵, 高分) |

#### 4. 巴菲特指标

| 项目 | 内容 |
|------|------|
| **名称** | 巴菲特指标 (buffett_ratio) |
| **计算公式** | `比值 = M2(亿) / A股总市值(亿) × 100` |
| **时间范围** | 近10年 |
| **数据来源** | tushare m2_monthly (月度) + stock_market_cap (日频总市值) |
| **计算逻辑** | M2/GDP的变体，用M2替代GDP更及时。衡量市场整体杠杆和泡沫程度 |
| **代表意义** | 比值高=资金充裕/泡沫=高分；比值低=资金紧张/低估=低分 |
| **数据量** | 2750天; M2最新: 353万亿 |
| **当前值** | 比值=2.68, 分位=81.5% |
| **分层含义** | 0-20%: 资金紧张(熊市) \| 40-60%: 中性 \| 80-100%: 资金充裕/泡沫(牛市) |

### 资金维度 (权重25%)

#### 5. 两融余额/流通市值比

| 项目 | 内容 |
|------|------|
| **名称** | 两融余额比 (margin_ratio) |
| **计算公式** | `比值 = (融资余额rzye + 融券余额rqye) / 全市场流通市值total_circ_mv × 100` |
| **时间范围** | 近504天 (2年滚动窗口) |
| **数据来源** | tushare margin_history + daily_circ_mv (预计算表) |
| **计算逻辑** | 衡量市场整体杠杆水平。对ratio自身做504天历史分位 |
| **代表意义** | 比值高=杠杆高=牛市特征=高分；比值低=杠杆低=熊市特征=低分 |
| **数据量** | 2587天融资数据; 预计算表2772天流通市值; 有效交叉2587天 |
| **当前值** | rzye=2.88万亿, rqye=209亿, circ_mv=34.27万亿, ratio=8.47%, 分位=99.2% |
| **分层含义** | 0-20%: 杠杆低(熊市, <1.5%) \| 40-60%: 中性 \| 80-100%: 杠杆高(牛市, >3%) |

#### 6. 北向资金方向

| 项目 | 内容 |
|------|------|
| **名称** | 北向资金方向 (northbound) |
| **计算公式** | `分位 = (历史250日净流入 < 当前250日净流入) / 历史窗口数 × 100` |
| **时间范围** | 近10年，250日滚动窗口 |
| **数据来源** | tushare northbound_history.north_net |
| **计算逻辑** | 取最近250日累计净流入金额，与历史各250日窗口比较分位 |
| **代表意义** | 分位高=外资持续流入=看好A股=高分；分位低=外资流出=看空=低分 |
| **数据量** | 2683天; 2433个250日窗口 |
| **当前值** | 当日净流入407410万, 250天分位=96.0% |
| **分层含义** | 0-20%: 外资大幅流出 \| 40-60%: 中性 \| 80-100%: 外资持续流入 |

#### 7. 北向资金累计流入分位

| 项目 | 内容 |
|------|------|
| **名称** | 北向累计流入分位 (northbound_cumflow) |
| **计算公式** | `分位 = (历史250日累计净流入 < 当前250日累计净流入) / 历史窗口数 × 100` |
| **时间范围** | 近10年，250日滚动窗口 |
| **数据来源** | tushare northbound_history.north_net |
| **计算逻辑** | 与northbound类似，但使用rolling sum计算累计值，更平滑 |
| **代表意义** | 反映外资中期配置力度，比单日方向更稳定 |
| **数据量** | 2683天; 2433个窗口 |
| **当前值** | 近250日累计=6656万, 10年分位=100.0% |
| **分层含义** | 0-20%: 外资累计大幅流出 \| 40-60%: 中性 \| 80-100%: 外资累计大幅流入 |

### 情绪维度 (权重20%)

#### 8. 换手率

| 项目 | 内容 |
|------|------|
| **名称** | 换手率 (turnover) |
| **计算公式** | `换手率 = 全市场成交额(千元) / 全市场流通市值(万元) × 10` |
| **时间范围** | 近10年 |
| **数据来源** | tushare stock_daily.amount + stock_daily.circ_mv + daily_turnover (预计算表) |
| **计算逻辑** | 注意amount单位千元, circ_mv单位万元。仅用tushare amount日期(>4000行) |
| **代表意义** | 换手率高=市场活跃=情绪高涨=高分；换手率低=市场冷淡=低分 |
| **数据量** | 预计算表: 1374天(仅tushare覆盖期); 最新: 2.99% |
| **当前值** | amount=27926亿, circ_mv=10449万亿, turnover=2.99%, 分位=99.8% |
| **分层含义** | 0-20%: 极度冷清(<0.8%) \| 40-60%: 正常(1-1.5%) \| 80-100%: 极度活跃(>3%) |

#### 9. 涨跌家数比

| 项目 | 内容 |
|------|------|
| **名称** | 涨跌家数比 (up_down_ratio) |
| **计算公式** | `比值 = 上涨家数 / 下跌家数` |
| **时间范围** | 近10年 |
| **数据来源** | tushare stock_daily.pct_change (全市场) + daily_updown (预计算表) |
| **计算逻辑** | 统计当日上涨/下跌股票数量比值，查10年历史分位 |
| **代表意义** | 比值高=普涨=全面牛市=高分；比值低=普跌=熊市=低分 |
| **数据量** | 预计算表: 2772天 |
| **当前值** | 上涨0.47, 下跌1.00, 比值=0.47, 分位=19.6% |
| **分层含义** | 0-20%: 普跌(熊市) \| 40-60%: 涨跌各半 \| 80-100%: 普涨(牛市) |

#### 10. 涨停占比

| 项目 | 内容 |
|------|------|
| **名称** | 涨停占比 (limit_up_ratio) |
| **计算公式** | `占比 = 涨停股票数(pct_change>=9.9%) / 有数据总股票数 × 100` |
| **时间范围** | 近10年 |
| **数据来源** | tushare stock_daily.pct_change + daily_limit (预计算表) |
| **计算逻辑** | 统计涨停股票占比，查10年历史分位 |
| **代表意义** | 占比高=极端乐观=情绪过热=高分 |
| **数据量** | 预计算表: 2772天; 最新: 103/5192=1.98% |
| **当前值** | limit_up_ratio=0.0198, 分位=79.2% |
| **分层含义** | 0-20%: 涨停极少 \| 40-60%: 正常 \| 80-100%: 涨停潮(情绪过热) |

#### 11. 涨跌停比

| 项目 | 内容 |
|------|------|
| **名称** | 涨跌停比 (limit_ratio) |
| **计算公式** | `比值 = 涨停数 / max(跌停数, 1)` (封顶10) |
| **时间范围** | 近10年 |
| **数据来源** | tushare stock_daily.pct_change + daily_limit (预计算表) |
| **计算逻辑** | 涨停数/跌停数，查10年历史分位 |
| **代表意义** | 比值高=涨停远多于跌停=情绪极度乐观=高分；比值低=跌停多=恐慌=低分 |
| **数据量** | 预计算表: 2772天; 最新: 103/13=7.92 |
| **当前值** | limit_ratio=7.92, 分位=93.2% |
| **分层含义** | 0-20%: 跌停远多于涨停(恐慌) \| 40-60%: 涨跌停接近 \| 80-100%: 涨停远多于跌停(狂热) |

### 技术维度 (权重10%)

| 项目 | 内容 |

| **计算逻辑** | 统计收盘价高于250日均线的股票占比，查10年历史分位 |
| **代表意义** | 比例高=多数股票趋势向上=牛市=高分；比例低=多数趋势向下=熊市=低分 |

| **当前值** | above_ma250_ratio=0.3884, 分位=43.6% |
| **分层含义** | 0-20%: 低于10%(熊市) \| 40-60%: 30-50%(震荡) \| 80-100%: 高于70%(牛市) |

#### 13. 均线偏离度

| 项目 | 内容 |

| **名称** | 均线偏离度 (deviation_ma250) |

| **代表意义** | 偏离大=价格远高于均线=泡沫风险=高分；偏离小/负=低估=低分 |
| **数据量** | 2771天 |
| **当前值** | 偏离度=3.84%, 分位=67.8% |
| **分层含义** | 0-20%: 低于均线(熊市) \| 40-60%: 接近均线 \| 80-100%: 远高于均线(泡沫) |

### 结构维度 (权重20%)

#### 14. 行业分化度 (连续版, 反向)

| 项目 | 内容 |
|------|------|
| **名称** | 行业分化度 (sector_divergence) |
| **计算公式** | `分化度 = 各行业平均涨跌幅的标准差(std)` |
| **时间范围** | 近10年 |
| **数据来源** | tushare stock_daily.pct_change + stock_industry.industry + daily_sector_div (预计算表) |
| **计算逻辑** | 按84个行业分组计算平均涨跌幅，取标准差，查10年历史分位。反向：std低=普涨=高分 |
| **代表意义** | 分化度低=普涨=全面牛市=高分；分化度高=结构性行情=低分 |
| **数据量** | 预计算表: 2772天 × 84行业; 最新: std=1.24 |
| **当前值** | sector_std=1.2399, 分位=42.8%(反向) |
| **分层含义** | 0-20%: 高分化(结构性行情) \| 40-60%: 中等 \| 80-100%: 低分化(全面普涨) |

#### 15. AH溢价指数

| 项目 | 内容 |
|------|------|
| **名称** | AH溢价指数 (ah_premium_index) |
| **计算公式** | `溢价 = 15只核心AH股的 A股价格/H股价格 的中位数` |
| **时间范围** | 近10年 |
| **数据来源** | H股: akshare stock_hk_daily; A股: tushare daily |
| **计算逻辑** | 取15只最大AH股，计算A股CNY价/H股HKD价比值，取中位数，查10年历史分位 |
| **代表意义** | 溢价高=A股相对H股贵=高分；溢价低=A股便宜=低分 |
| **数据量** | ah_premium表: 2688天 × 15只AH股; 最新: 1.1074 |
| **当前值** | A/H比值=1.1074, 10年分位=41.6% |
| **分层含义** | 0-20%: A股比H股便宜 \| 40-60%: 接近平价 \| 80-100%: A股远贵于H股(牛市特征) |

## 三、综合得分计算

### 3.1 维度内合成

每个维度内的子指标等权合成。若某子指标异常(>3σ)或为0/None则舍弃，其余重新等权归一。

### 3.2 维度间加权合成

| 维度 | 权重 | 依据 |
|------|------|------|
| 估值 | 25% | 区分度53.2, 牛市顶/熊底信号最可靠 |
| 资金 | 25% | 区分度86.1(最佳) |
| 情绪 | 20% | 区分度60.8 |
| 技术 | 10% | 区分度27.7(最差), 降权 |
| 结构 | 20% | 区分度36.1 |

### 3.3 热度区间

| 颜色 | 分数 | 含义 | 行动建议 |
|------|------|------|---------|
| 🟢 绿色安全 | 0–40 | 估值合理/偏低，情绪冷淡 | 减仓信号远 |
| 🟡 黄色警惕 | 40–70 | 部分指标偏高 | 需关注，谨慎操作 |
| 🔴 红色预警 | 70–100 | 多项指标历史高位 | 考虑减仓/离场 |

### 3.4 防抖机制

- 红区连续2天才发"进入红区"飞书通知
- 恢复1天后发"脱离红区"通知
- 黄区/绿区变化不触发通知
