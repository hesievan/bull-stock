# A股牛市热度指数 — 指标详细说明

> 生成时间: 2026-06-03 | 指标版本: v3.0 (5维度 16子指标)

---

## 一、指标体系总览

| 维度 | 权重 | 子指标数 | 子指标 |
|------|------|---------|--------|
| 估值 | 25% | 4 | PE分位, PB分位, 破净率, 巴菲特指标 |
| 资金 | 25% | 3 | 两融余额比, 北向资金方向, 北向累计流入分位 |
| 情绪 | 20% | 4 | 换手率, 涨跌家数比, 涨停占比, 涨跌停比 |
| 技术 | 10% | 2 | 站上年线比例, 均线偏离度 |
| 结构 | 20% | 2 | 行业分化度(连续版), AH溢价指数 |

**综合得分 = 估值×25% + 资金×25% + 情绪×20% + 技术×10% + 结构×20%**

## 二、各指标详细说明

### 估值维度 (权重25%)

#### 1. PE历史分位

| 项目 | 内容 |
|------|------|
| **名称** | PE历史分位 (PE_percentile) |
| **计算公式** | `PE分位 = (历史PE中位数 < 当前PE中位数) / 历史样本数 × 100` |
| **时间范围** | 近10年 (2015-01-05 ~ 2026-06-02) |
| **数据来源** | baostock(stock_daily.peTTM) + tushare(daily_basic.pe_ttm) |
| **计算逻辑** | 沪深300+中证500成分股口径，用历史成分股截面(月末)计算PE中位数，查预计算表index_daily_pe |
| **代表意义** | 衡量当前PE在历史上的相对位置。高分=估值偏贵=牛市顶峰特征 |
| **数据量** | 2750个交易日PE中位数 |
| **当前值** | PE中位数=25.96, 分位=58.4% |
| **分层含义** | 0-20%: 极度低估(熊底) | 20-40%: 偏低 | 40-60%: 中性 | 60-80%: 偏高 | 80-100%: 极度高估(牛市顶) |

#### 2. PB历史分位

| 项目 | 内容 |
|------|------|
| **名称** | PB历史分位 (PB_percentile) |
| **计算公式** | `PB分位 = (历史PB中位数 < 当前PB中位数) / 历史样本数 × 100` |
| **时间范围** | 近10年 (2015-01-05 ~ 2026-06-02) |
| **数据来源** | baostock(stock_daily.pbMRQ) + tushare(daily_basic.pb_mrq) |
| **计算逻辑** | 同PE，成分股口径PB中位数的历史分位 |
| **代表意义** | 衡量当前PB在历史上的相对位置。与PE互补，PB更稳定 |
| **数据量** | 2750个交易日PB中位数 |
| **当前值** | PB中位数=2.20, 分位=60.1% |
| **分层含义** | 0-20%: 极度低估 | 20-40%: 偏低 | 40-60%: 中性 | 60-80%: 偏高 | 80-100%: 极度高估 |

#### 3. 破净率 (反向)

| 项目 | 内容 |
|------|------|
| **名称** | 破净率 (below_net_rate) |
| **计算公式** | `破净率 = PB<1的股票数 / 有PB数据的总股票数 × 100` |
| **时间范围** | 近10年 |
| **数据来源** | stock_daily.pbMRQ (全市场) |
| **计算逻辑** | 全市场口径，统计PB<1的股票占比，然后用反向分位：破净率越高=越便宜=得分越低 |
| **代表意义** | 破净率高=市场便宜=低分；破净率低=市场贵=高分 |
| **数据量** | 5151只有PB数据的股票 |
| **当前值** | 351/5151 = 6.8%, 分位=39.4%(反向) |
| **分层含义** | 0-20%: 破净率高(便宜) | 40-60%: 中性 | 80-100%: 破净率低(贵) |

#### 4. 巴菲特指标

| 项目 | 内容 |
|------|------|
| **名称** | 巴菲特指标 (buffett_ratio) |
| **计算公式** | `比值 = M2(亿) / A股总市值(亿) × 100` |
| **时间范围** | 近10年 |
| **数据来源** | M2: tushare(m2_monthly)月度; 总市值: tushare(stock_market_cap)日频 |
| **计算逻辑** | M2/GDP的变体，用M2替代GDP更及时。衡量市场整体杠杆和泡沫程度 |
| **代表意义** | 比值高=资金充裕/泡沫=高分；比值低=资金紧张/低估=低分 |
| **数据量** | 2750个交易日 |
| **当前值** | M2=353万亿, 总市值=132万亿, 比值=267.7%, 分位=81.5% |
| **分层含义** | 0-20%: 资金紧张(熊市) | 40-60%: 中性 | 80-100%: 资金充裕/泡沫(牛市) |

### 资金维度 (权重25%)

#### 5. 两融余额/流通市值比

| 项目 | 内容 |
|------|------|
| **名称** | 两融余额比 (margin_ratio) |
| **计算公式** | `比值 = (融资余额rzye + 融券余额rqye) / 全市场流通市值total_circ_mv × 100` |
| **时间范围** | 近2年 (504天窗口) |
| **数据来源** | 融资融券: tushare(margin_history); 流通市值: stock_daily.circ_mv |
| **计算逻辑** | 衡量市场整体杠杆水平。对ratio自身做504天历史分位 |
| **代表意义** | 比值高=杠杆高=牛市特征=高分；比值低=杠杆低=熊市特征=低分 |
| **数据量** | 2587天融资数据 × 2771天流通市值 |
| **当前值** | (28834亿+209亿)/34万亿 = 8.47%, 504天分位=100% |
| **分层含义** | 0-20%: 杠杆低(熊市3%以下) | 40-60%: 中性 | 80-100%: 杠杆高(牛市8%以上) |

#### 6. 北向资金方向

| 项目 | 内容 |
|------|------|
| **名称** | 北向资金方向 (northbound) |
| **计算公式** | `分位 = (历史250日净流入 < 当前250日净流入) / 历史窗口数 × 100` |
| **时间范围** | 近10年，250日滚动窗口 |
| **数据来源** | tushare(moneyflow_hsgt.north_net) |
| **计算逻辑** | 取最近250日累计净流入金额，与历史各250日窗口比较分位 |
| **代表意义** | 分位高=外资持续流入=看好A股=高分；分位低=外资流出=看空=低分 |
| **数据量** | 2683天北向资金数据，2433个250日窗口 |
| **当前值** | 当日净流入407410万, 250天分位=96.0% |
| **分层含义** | 0-20%: 外资大幅流出 | 40-60%: 中性 | 80-100%: 外资持续流入 |

#### 7. 北向资金累计流入分位

| 项目 | 内容 |
|------|------|
| **名称** | 北向累计流入分位 (northbound_cumflow) |
| **计算公式** | `分位 = (历史250日累计净流入 < 当前250日累计净流入) / 历史窗口数 × 100` |
| **时间范围** | 近10年，250日滚动窗口 |
| **数据来源** | tushare(moneyflow_hsgt.north_net) |
| **计算逻辑** | 与northbound类似，但使用rolling sum计算累计值，更平滑 |
| **代表意义** | 反映外资中期配置力度，比单日方向更稳定 |
| **数据量** | 2683天，2433个窗口 |
| **当前值** | 近250日累计=6656万, 10年分位=99.95% |
| **分层含义** | 0-20%: 外资累计大幅流出 | 40-60%: 中性 | 80-100%: 外资累计大幅流入 |

### 情绪维度 (权重20%)

#### 8. 换手率

| 项目 | 内容 |
|------|------|
| **名称** | 换手率 (turnover) |
| **计算公式** | `换手率 = 全市场成交额(元) / 全市场流通市值(元) × 100` |
| **时间范围** | 近10年 |
| **数据来源** | 成交额: baostock(stock_daily.amount); 流通市值: tushare(stock_daily.circ_mv) |
| **计算逻辑** | 注意：amount和circ_mv在stock_daily中互斥(baostock只提供amount, tushare只提供circ_mv)，分别求和再除 |
| **代表意义** | 换手率高=市场活跃=情绪高涨=高分；换手率低=市场冷淡=低分 |
| **数据量** | 2771个交易日 |
| **当前值** | 3.95%, 10年分位=99.7% |
| **分层含义** | 0-20%: 极度冷清(1%以下) | 40-60%: 正常(1-2%) | 80-100%: 极度活跃(3%以上) |

#### 9. 涨跌家数比

| 项目 | 内容 |
|------|------|
| **名称** | 涨跌家数比 (up_down_ratio) |
| **计算公式** | `比值 = 上涨家数 / 下跌家数` |
| **时间范围** | 近10年 |
| **数据来源** | stock_daily.pct_change (全市场) |
| **计算逻辑** | 统计当日上涨/下跌股票数量比值，查10年历史分位 |
| **代表意义** | 比值高=普涨=全面牛市=高分；比值低=普跌=熊市=低分 |
| **数据量** | 2771天 × ~5200只/天 |
| **当前值** | 1373/3742 = 0.37, 10年分位=19.6% |
| **分层含义** | 0-20%: 普跌(熊市) | 40-60%: 涨跌各半 | 80-100%: 普涨(牛市) |

#### 10. 涨停占比

| 项目 | 内容 |
|------|------|
| **名称** | 涨停占比 (limit_up_ratio) |
| **计算公式** | `占比 = 涨停股票数(pct_change>=9.9%) / 有数据总股票数 × 100` |
| **时间范围** | 近10年 |
| **数据来源** | stock_daily.pct_change |
| **计算逻辑** | 统计涨停股票占比，查10年历史分位 |
| **代表意义** | 占比高=极端乐观=情绪过热=高分 |
| **数据量** | 2771天 × ~5200只/天 |
| **当前值** | 103/5151 = 2.00%, 10年分位=79.2% |
| **分层含义** | 0-20%: 涨停极少 | 40-60%: 正常 | 80-100%: 涨停潮(情绪过热) |

#### 11. 涨跌停比

| 项目 | 内容 |
|------|------|
| **名称** | 涨跌停比 (limit_ratio) |
| **计算公式** | `比值 = 涨停数 / max(跌停数, 1)` (封顶10) |
| **时间范围** | 近10年 |
| **数据来源** | stock_daily.pct_change |
| **计算逻辑** | 涨停数/跌停数，查10年历史分位 |
| **代表意义** | 比值高=涨停远多于跌停=情绪极度乐观=高分；比值低=跌停多=恐慌=低分 |
| **数据量** | 2771天 × ~5200只/天 |
| **当前值** | 103/21 = 4.9, 10年分位=38.9% |
| **分层含义** | 0-20%: 跌停远多于涨停(恐慌) | 40-60%: 涨跌停接近 | 80-100%: 涨停远多于跌停(狂热) |

### 技术维度 (权重10%)

#### 12. 站上年线比例

| 项目 | 内容 |
|------|------|
| **名称** | 站上年线比例 (above_ma250_ratio) |
| **计算公式** | `比例 = close > MA250 的股票数 / 有数据总股票数 × 100` |
| **时间范围** | 近10年 |
| **数据来源** | stock_daily.close (计算250日均线) |
| **计算逻辑** | 统计收盘价高于250日均线的股票占比，查10年历史分位 |
| **代表意义** | 比例高=多数股票趋势向上=牛市=高分；比例低=多数趋势向下=熊市=低分 |
| **数据量** | 2771天 × ~5200只/天 |
| **当前值** | 43.6%, 10年分位=43.6% |
| **分层含义** | 0-20%: 低于10%(熊市) | 40-60%: 30-50%(震荡) | 80-100%: 高于70%(牛市) |

#### 13. 均线偏离度

| 项目 | 内容 |
|------|------|
| **名称** | 均线偏离度 (deviation_ma250) |
| **计算公式** | `偏离度 = (上证综指收盘价 / MA250 - 1) × 100` |
| **时间范围** | 近10年 |
| **数据来源** | baostock(index_daily.sh000001) |
| **计算逻辑** | 上证综指偏离250日均线的百分比，查10年历史分位 |
| **代表意义** | 偏离大=价格远高于均线=泡沫风险=高分；偏离小/负=低估=低分 |
| **数据量** | 2771天 |
| **当前值** | 偏离度=67.8%, 10年分位=67.8% |
| **分层含义** | 0-20%: 低于均线(熊市) | 40-60%: 接近均线 | 80-100%: 远高于均线(泡沫) |

### 结构维度 (权重20%)

#### 14. 行业分化度 (连续版, 反向)

| 项目 | 内容 |
|------|------|
| **名称** | 行业分化度 (sector_divergence) |
| **计算公式** | `分化度 = 各行业平均涨跌幅的标准差(std)` |
| **时间范围** | 近10年 |
| **数据来源** | stock_daily.pct_change + stock_industry.industry |
| **计算逻辑** | 按行业分组计算平均涨跌幅，取标准差，查10年历史分位。反向：std低=普涨=高分 |
| **代表意义** | 分化度低=普涨=全面牛市=高分；分化度高=结构性行情=低分 |
| **数据量** | 2771天 × 83个行业 |
| **当前值** | std=1.27, 10年分位=38.9%(反向) |
| **分层含义** | 0-20%: 高分化(结构性行情) | 40-60%: 中等 | 80-100%: 低分化(全面普涨) |

#### 15. AH溢价指数

| 项目 | 内容 |
|------|------|
| **名称** | AH溢价指数 (ah_premium_index) |
| **计算公式** | `溢价 = 15只核心AH股的 A股价格/H股价格 的中位数` |
| **时间范围** | 近10年 |
| **数据来源** | H股: akshare(stock_hk_daily); A股: baostock(stock_daily.close) |

| **代表意义** | 溢价高=A股相对H股贵=高分；溢价低=A股便宜=低分 |
| **数据量** | 2687天 × 15只AH股 |
| **当前值** | A/H比值=1.0887, 10年分位=41.6% |
| **分层含义** | 0-20%: A股比H股便宜 | 40-60%: 接近平价 | 80-100%: A股远贵于H股(牛市特征) |

## 三、综合得分计算

### 3.1 维度内合成

每个维度内的子指标等权合成。若某子指标异常(>3σ)或为0/None则舍弃，其余重新等权归一。

**示例 (估值维度)**:
```
估值得分 = (PE分位 + PB分位 + 破净率反向分位 + 巴菲特指标分位) / 4
         = (58.4 + 60.1 + 39.4 + 81.5) / 4
         = 59.9
```

### 3.2 维度间加权合成

```
综合得分 = 估值×25% + 资金×25% + 情绪×20% + 技术×10% + 结构×20%
        = 59.9×0.25 + 98.7×0.25 + 59.3×0.20 + 55.7×0.10 + 40.3×0.20
        = 14.98 + 24.68 + 11.86 + 5.57 + 8.06
        = 65.1
```

### 3.3 热度区间

| 颜色 | 分数 | 含义 | 行动建议 |
|------|------|------|---------|
| 🟢 绿色安全 | 0–40 | 估值合理/偏低，情绪冷淡 | 减仓信号远 |
| 🟡 黄色警惕 | 40–70 | 部分指标偏高 | 需关注，谨慎操作 |
| 🔴 红色预警 | 70–100 | 多项指标历史高位 | 考虑减仓/离场 |

### 3.4 防抖机制

- 红区连续2天才发"进入红区"飞书通知
- 恢复1天后发"脱离红区"通知
- 黄区/绿区变化不触发通知
